# Captcha Background SDK 官网

产品官网，使用 TypeScript + React + Ant Design 开发。

## 技术栈

- React 18
- TypeScript
- Vite
- Ant Design 5
- React Router v6

## 页面结构

- `/` - 首页（产品介绍、特性、代码示例）
- `/features` - 功能特性
- `/sdk` - SDK 文档（Python + TypeScript）
- `/docs` - 文档中心
- `/download` - 下载页面

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建输出在 `dist/` 目录。

## 部署

可以部署到 GitHub Pages 或其他静态托管服务。
