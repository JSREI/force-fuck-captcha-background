# Electron UI Tool

验证码辅助桌面工具（Electron + React + TypeScript）。

## 运行

```bash
cd apps/electron-ui
npm install
npm run dev
```

## 构建

```bash
cd apps/electron-ui
npm run build:webpack
npm run build
```

## 目录

```text
apps/electron-ui/
├── src/
│   ├── main/
│   └── renderer/
├── package.json
├── tsconfig.json
└── webpack.config.js
```
